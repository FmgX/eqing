Linux 安装Resilio Sync

Resilio Sync （原 BTsync）是由 Resilio 公司开发的一种端到端（P2P）文件同步软件。该软件在几乎所有平台上都有官方版本，包括桌面端 Windows/Mac/Linux 版、移动端 Android/iOS 版，以及 NAS 版等。

安装起来也十分的简单

下载Sync

官方网站：https://www.resilio.com/
客户端下载：https://www.resilio.com/platforms/desktop/
可以在本地电脑上下载后通过winscp上传到你想安装到的Linux主机上面，也可以直接使用wget命令直接下载到Linux主机上
安装前需要判定Linux的位数来决定下载哪个版本

uname -m   //如果返回 x86_64 那就是64位，如果返回 i386或386 那就是32位
//64位
wget --no-check-certificate -O sync.tar.gz https://download-cdn.resilio.com/stable/linux-x64/resilio-sync_x64.tar.gz
//32位
wget --no-check-certificate -O sync.tar.gz https://download-cdn.resilio.com/stable/linux-i386/resilio-sync_i386.tar.gz

安装

//解压上面下载的并给予限权
tar -xzf sync.tar.gz && rm -rf sync.tar.gz
chmod +x rslsync

设置时间

如果你的时间没有问题请忽略此步

因为sync是同步软件所以对时间有要求

//修改主机的时区为上海北京
cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

随后参考Linux设置时间

启动

上面命令执行完并没有报错，就可以启动了
记得放行,端口放行请参考linux端口问题

//监听端口为8888，
./rslsync --webui.listen 0.0.0.0:8888
