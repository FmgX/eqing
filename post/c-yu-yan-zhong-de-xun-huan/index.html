C语言中的循环

1. while

while 表达式是顶部驱动（top-driven）的循环
也就是先计算循环条件（也就是控制表达式）。如果为 真或者1，就执行循环体。否则退出循环
格式为

while （循环条件）
  {
      循环体
  }

eg.

int sum=0,i=1,N=5;    //使用初始条件时需要定义
while(i<=N)     
{
     sum=sum+i;
     i++;
}

2. do while

do...while 循环是一种底部驱动的循环
也就是在计算循环条件（也就是控制表达式）之前先执行循环体，随后在计算循环条件（也就是控制表达式）如果为 真或者1，就返回再次执行循环体。否则退出循环
与while区别在于，开头无论什么都先执行一次do
格式

do
{
   循环体
} while (循环条件)

4. for

for 循环也是一个顶部驱动的循环
格式为：

for ([表达式1];[表达式2];[表达式3])
{
   循环体
}
  1. 表达式1——初始条件
    在循环体开始前执行
  2. 表达式2——循环条件（控制表达式）
    语句 2 定义运行循环体的条件
  3. 表达式3——调节器（eg.上面while例句中的i++）
    语句 3 在循环体已被执行之后执行
