Linux挂载硬盘

手里一直有台Gi945，前一段时间正好有使用Resilio Sync的需求。不想使用主力机，正好用闲置已久Gi945，安装好centos 32位系统后无法在找到硬盘。需要挂载。遂有此篇,用以记录。机子已经拆了没有环境只记录步骤。

格式化分区

# 格式化整个磁盘
mkfs.ext3 /dev/sdb

# 格式化单个分区
mkfs.ext3 /dev/sdb1

挂载使用

# 新建一个挂载点
mkdir /data 

# 挂载
mount /dev/sdb1 /data

开机自动挂载

在vi /etc/fstab中添加新硬盘的挂载信息，添加下面一行:
/dev/sdb1 /data ext3 defaults 1 2（如果还有一个分区就是1 3，以此类推)
这样,每次开机后,系统会自动将/dev/sdb1挂载到/data
