使用DDNS动态解析ipv6

在ipv4枯竭的情况下免费得到一个公网ipv4是几乎不可能了，不过得到公网Ipv6却十分容易。
本文总结一下近来使用DDNS解析ipv6的办法。

1.准备

域名（哪里注册的都可以）、一个可以的获取到ipv6的网络环境（主要是在光猫和路由器上，现在大部分地区都支持ipv6）、
这里使用的是老毛子的路由固件，好在集成了各种DDNS工具你可以使用电脑或者其他设备。使用的是阿里DDNS其他的DDNS平台都是可以的

2.使用

只用填入Access Key ID 、Access Key Secret、域名即可 。使用ipv6则填入顶级域名3[ipv6]，如果有ipv4的就填入1，2位置
下面讲解获取Access Key ID 、Access Key Secret的步骤，至于域名花钱买个也可以免费注册也可以，解析到对应的DDNS平台就行

使用阿里DDNS就需要将你的域名解析到阿里DDNS，不过我的就是在阿里注册的可以省略。
打开阿里域名控制台添加你想使用的域名,
eg.

鼠标放到头像上打开AccessKey 管理

使用子用户比较安全些

创建用户


创建好后点击你创建好的用户，点创建 AccessKey，随后复制好你的Access Key ID 、Access Key Secret

到这里就结束了，只用把你的Access Key ID 、Access Key Secret、域名一一填入对应位置。
